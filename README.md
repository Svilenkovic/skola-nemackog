# Škola Nemačkog — German Language Courses

![Status](https://img.shields.io/badge/status-live-brightgreen) ![PHP](https://img.shields.io/badge/PHP-8.3-777BB4)

## 🔗 Live: [skolanemackog.online](https://skolanemackog.online)

## Overview
Website for an online German language school offering courses from A1 to C1 level. Features course catalog, pricing, and free consultation booking.

## Tech Stack
- **Backend:** PHP 8.3
- **Frontend:** Responsive CSS3
- **Hosting:** Nginx, Ubuntu 24.04

## Features
- 📚 Course catalog (A1-C1 levels)
- 💰 Pricing from 22,000 RSD
- 📞 Free consultation booking
- 👩‍🏫 Teacher profiles
- 📱 Responsive design
